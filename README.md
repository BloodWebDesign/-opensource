это opensource проект от BLOOD Web Design.

Разрешается:
Распростронять репозиторий
Редактировать и публиковать репозиторий

Чтобы начать редактирование откройте файл index.html через visual studio code или иные программы.
Чтобы просмотреть проект - ctrl + f5 или запускаете файл index.html через браузер.
Чтобы изменения вступили в силу - ctrl + s.
